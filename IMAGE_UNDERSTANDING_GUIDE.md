# Agent Mode 图像理解功能使用指南

## 📸 功能概述

Agent Mode 现在支持**图像理解**功能！你可以：

1. **仅凭图片创作故事** - 上传一张图片，AI 会根据图片内容生成完整的故事草案
2. **图片 + 文字混合输入** - 提供图片和文字描述，AI 会结合两者生成更精准的故事
3. **视觉元素提取** - AI 会自动识别图片中的：
   - 人物特征（外貌、服装、表情）
   - 场景元素（环境、光线、色彩）
   - 情绪氛围（快乐、悲伤、紧张等）
   - 动作和姿态

## 🚀 使用方法

### 方法一：仅使用图片

1. 打开 Agent Mode（点击"Agent"按钮）
2. 在"故事模板"字段留空
3. 在"参考图片"字段输入图片 URL
4. 点击"生成草案"

**示例图片 URL：**
```
https://ark-project.tos-cn-beijing.volces.com/doc_image/ark_demo_img_1.png
```

### 方法二：图片 + 文字描述

1. 在"故事模板"字段输入简短描述
2. 在"参考图片"字段输入图片 URL
3. 点击"生成草案"

**示例：**
- 故事模板："一个关于勇气的故事"
- 参考图片：一张冒险者攀登高山的图片

### 方法三：纯文字（原有功能）

如果不提供图片 URL，系统会使用纯文字模式，就像之前一样工作。

## 🔧 API 技术细节

### API 模型

使用 **doubao-seed-2-0-pro-260215** 模型（支持图像理解的版本）

### API 端点

```
POST https://ark.cn-beijing.volces.com/api/v3/responses
```

### 请求格式

```json
{
```

## 📋 AI 如何处理图片

### 1. 视觉元素识别

AI 会观察图片中的：

**人物：**
- 性别、年龄、种族
- 发色、发型、面部特征
- 服装风格、配饰
- 表情、情绪、姿态

**场景：**
- 环境（室内/室外、城市/自然）
- 光线（明亮/昏暗、暖色/冷色）
- 色彩（主色调、对比度）
- 道具和物品

**氛围：**
- 情绪（快乐、悲伤、紧张、温馨）
- 氛围（神秘、浪漫、恐怖、励志）
- 风格（写实、卡通、复古、未来）

### 2. 故事生成策略

AI 会根据图片内容：

**角色设计：**
- 提取人物特征 → `appearancePrompt`
- 推断性格和背景 → `bio`
- 保持视觉一致性 → `basePrompt`

**场景设计：**
- 匹配图片视觉风格 → `videoPrompt`
- 提取关键视觉元素
- 设计合适的镜头（景别、角度、运镜）

**故事结构：**
- 根据图片氛围设定基调
- 设计符合视觉风格的剧情
- 创建有意义的分支选择

### 3. 示例转换

**输入图片：** 一个年轻女孩在雨中哭泣

**AI 生成：**

```json
{
  "characters": [
    {
      "id": "girl",
      "name": "小雨",
      "bio": "18岁，高中生，黑色长发，穿校服，性格敏感内向",
      "appearancePrompt": "亚洲女孩，18岁，黑色长发湿漉漉，校服淋湿，眼泪在脸上",
      "basePrompt": "女孩，黑色长发，校服，哭泣"
    }
  ],
  "scenes": [
    {
      "id": "s1",
      "title": "雨中哭泣",
      "summary": "小雨在雨中哭泣，情绪崩溃",
      "videoPrompt": "雨中街道，近景，女孩低头哭泣，雨水打在脸上，冷色调，城市灯光模糊"
    }
  ]
}
```

## 🎯 最佳实践

### 1. 选择合适的图片

**推荐：**
- ✅ 人物清晰的图片
- ✅ 场景明确的图片
- ✅ 有情绪氛围的图片
- ✅ 光线良好的图片

**不推荐：**
- ❌ 模糊不清的图片
- ❌ 过于复杂的图片（元素太多）
- ❌ 纯文字图片
- ❌ 低分辨率图片

### 2. 图片 URL 要求

- 必须是公网可访问的 URL
- 支持格式：PNG、JPG、JPEG
- 建议分辨率：512x512 ~ 1920x1080
- 建议文件大小：< 5MB

### 3. 结合文字描述

虽然 AI 可以仅凭图片生成故事，但提供简短的文字描述会帮助 AI：

**示例 1：**
- 图片：一张科幻城市的图片
- 文字："一个关于科技与人性的故事"

**示例 2：**
- 图片：两个人争吵的图片
- 文字："展现冲突的两种解决方式：暴力 vs 对话"

### 4. 使用反馈优化

如果第一版生成不满意，在"给 Agent 的意见"中提供具体反馈：

**示例反馈：**
```
"角色太大众化，请让主角更有特色"
"场景不够紧张，请增加悬念感"
"分支选择不够明确，请让两个选项差异更明显"
```

## 💡 创意玩法

### 1. 风格参考

上传一张电影截图或艺术作品，让 AI 参考其视觉风格生成故事。

### 2. 角色设定

上传人物肖像，让 AI 基于该角色设计完整的故事。

### 3. 场景扩展

上传一个场景图片，让 AI 构思可能发生在这个场景的故事。

### 4. 情绪引导

上传有强烈情绪氛围的图片，让 AI 生成相应情感基调的故事。

## 🔍 技术限制

1. **图片大小**：API 对图片大小有限制，建议 < 5MB
2. **超时时间**：图像处理需要更长时间，已将超时设置为 18 秒
3. **理解准确度**：AI 可能误解某些视觉元素，可以通过反馈修正
4. **URL 限制**：必须是公网可访问的 URL，不支持本地文件

## 📊 与纯文字模式的对比

| 特性 | 纯文字模式 | 图像理解模式 |
|------|----------|------------|
| 输入方式 | 文字描述 | 图片 + 可选文字 |
| 角色一致性 | 依赖文字描述 | 直接从图片提取 |
| 视觉风格 | 需要详细描述 | 自动识别 |
| 生成速度 | 更快（12秒） | 稍慢（18秒） |
| 视觉准确性 | 依赖提示词质量 | 更直观准确 |
| 使用场景 | 已有明确构思 | 有视觉参考或灵感 |

## 🎬 完整工作流程

1. **准备图片**：上传到图床，获取公网 URL
2. **打开 Agent Mode**：点击编辑器中的"Agent"按钮
3. **输入内容**：
   - （可选）故事模板：简短描述
   - 参考图片：输入图片 URL
   - （可选）给 Agent 的意见：具体要求
4. **生成草案**：点击"生成草案"按钮
5. **查看结果**：在右侧预览窗格查看生成的角色、场景、分支
6. **优化迭代**：如果不满意，在"给 Agent 的意见"中提供反馈，点击"根据意见重算"
7. **生成剧本**：满意后，点击"生成剧本"获取详细剧本
8. **应用到画布**：点击"应用到画布"将故事导入编辑器
