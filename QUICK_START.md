## 🎯 立即开始使用

### 步骤 1：配置 Agent API Key

1. 打开编辑器：http://localhost:3000/editor
2. 点击左上角的"API 设置"按钮
3. 在"Agent API 配置"部分输入你的 API Key
4. API Key 会自动保存到浏览器

**获取 API Key：**
- 访问：https://www.volcengine.com/docs/ark
- 注册/登录火山引擎账号
- 在方舟控制台获取 API Key

### 步骤 2：使用 Agent Mode 生成故事

**方式 A：纯文字模式**
1. 点击"Agent"按钮打开 Agent Mode
2. 在"故事模板"中输入故事描述
   - 示例："一个程序员发现 Bug，必须在截止前修复"
3. 点击"生成草案"

**方式 B：图片 + 文字模式**
1. 点击"Agent"按钮
2. 在"故事模板"中输入简短描述（可选）
3. 在"参考图片"中输入图片 URL
   - 示例：`https://ark-project.tos-cn-beijing.volces.com/doc_image/ark_demo_img_1.png`
4. 点击"生成草案"

**方式 C：纯图片模式**
1. 点击"Agent"按钮
2. "故事模板"留空
3. 在"参考图片"中输入图片 URL
4. AI 会根据图片自动创作故事

### 步骤 3：查看和优化草案

生成后，你可以看到：
- **角色**：详细的 bio 和 appearancePrompt
- **场景**：每个场景的镜头设计（景别、角度、运镜）
- **分支**：有意义的分支选择
- **连接**：场景之间的关系

如果不满意：
1. 在"给 Agent 的意见"中输入反馈
   - 示例："把主角塑造成更冷静"、"增加一个失败分支"
2. 点击"根据意见重算"

### 步骤 4：生成详细剧本

满意草案后：
1. 点击"生成剧本"
2. AI 会生成专业的影视剧本，包含：
   - 镜头语言（景别、角度、运镜）
   - 节奏控制（快慢交替）
   - 视觉描述（具体可感）
   - 情绪视觉化

### 步骤 5：应用到画布

1. 点击"应用到画布"
2. 故事会导入到编辑器
3. 可以进一步编辑和生成视频

## 🎨 Agent Prompts 优化详情

### Draft Agent 改进

**角色设计：**
- bio：年龄、职业、外貌、服装、性格、背景
- appearancePrompt：发色发型、五官、服装、配饰、体态
- basePrompt：视觉识别度

**镜头设计：**
- 景别：远景、全景、中景、近景、特写
- 角度：平视、俯视、仰视、荷兰角
- 运镜：固定、推拉、摇镜、跟拍、手持、无人机

**场景拆解：**
- 每个场景 3-8 秒
- 一个场景只讲一件事
- 四要素：谁 + 在哪 + 做什么 + 结果如何

### Script Agent 改进

**镜头语言系统：**
- 5 种景别的详细运用指南
- 4 种角度的情感表达
- 6 种运镜方式的场景适配

**节奏控制：**
- 快节奏：短镜头（1-2秒）、快速剪辑
- 慢节奏：长镜头（5-8秒）、缓慢运镜
- 节奏变化：冲突处加快，情感处放慢

**视觉描述技巧：**
- 具体化原则：不是"房间很乱"，而是"衣服堆在椅子上"
- 情绪视觉化：焦虑→频繁看表，悲伤→低头、雨景
- 感官细节：视觉、听觉、触觉

## 📸 图像理解功能

### 支持的场景

1. **仅凭图片创作**
   - 输入图片 URL
   - AI 观察并生成完整故事

2. **图片 + 文字**
   - 提供简短描述 + 图片
   - AI 结合两者生成精准故事

3. **视觉参考**
   - 上传风格参考图
   - AI 匹配视觉风格生成故事

### AI 如何处理图片

**识别内容：**
- 人物：性别、年龄、外貌、表情、姿态
- 场景：环境、光线、色彩、道具
- 氛围：情绪、风格、基调

**融入故事：**
- 角色的 appearancePrompt 匹配图片特征
- 场景的 videoPrompt 参考视觉风格
- 故事氛围匹配图片情绪

## 🔧 配置说明

### 环境变量（可选）

如果不想使用 UI 配置，可以创建 `.env.local` 文件：

```bash
# Agent API 配置
AGENT_API_KEY=你的API_Key
AGENT_API_MODEL=doubao-seed-2-0-pro-260215

# 视频生成 API 配置
NEXT_PUBLIC_VOLCENGINE_API_KEY=你的API_Key
```

### API Key 优先级

系统会按以下顺序查找 API Key：
1. UI 配置的 Agent API Key（最高优先级）
2. 环境变量 `AGENT_API_KEY`
3. 环境变量 `NEXT_PUBLIC_VOLCENGINE_API_KEY`

## 🐛 常见问题

**Q: Agent 生成失败？**
A: 检查 API Key 是否配置正确，访问 https://www.volcengine.com/docs/ark 获取

**Q: 图片无法识别？**
A: 确保图片 URL 可公网访问，格式为 PNG/JPG/JPEG

**Q: 生成速度慢？**
A: 图像理解需要 18 秒，纯文字需要 12 秒，请耐心等待

**Q: 如何查看保存的 API Key？**
A: 打开浏览器开发者工具 → Application → Local Storage → 搜索 `agent-api-key`
